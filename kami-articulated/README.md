# kami-articulated

URDF / MJCF / USD physics loader → kami-genesis articulation.

**Status**: R1.0 path reservation (ADR-2605261800).

## Scope

- URDF parser (ROS robot description)
- MJCF parser (MuJoCo XML)
- USD physics schema parser (via kami-usd)
- Unified `ArticulatedSystem` → kami-genesis World
